# Digital Notes - 项目管理文档

## 概述

基于 qiaomu-blog-opensource 开源博客模板搭建电子资料付费解锁站。

## 技术栈

- Next.js 16 + React 19 + TypeScript
- Cloudflare Workers / D1 / R2
- Stripe Checkout (Card + WeChat Pay)
- Resend (邮件)
